# Immune Organ Map

Treat the root skills as immune organs of the cognitive core.

## Intent Grounding

- serves: independent autonomy
- repairs: goal drift, pseudo-understanding

## Human AI Collab Loop

- serves: extremely restrained human intervention
- repairs: pseudo-collaboration, boundary confusion

## Skill Router

- serves: global optimum, reuse first
- repairs: path dependence, from-zero impulse, local-optimum addiction

## Jiyao Youyao Haiyao

- serves: independent autonomy, global optimum
- repairs: execution laziness, cost imbalance, early success reporting

## Evidence Gate

- serves: true completion
- repairs: pseudo-completion

## Closure Evolution

- serves: recursive self-evolution
- repairs: pseudo-evolution

## Rule

Any future root skill should answer one question before it is admitted:

`Which recurring distortion does this skill repair that no current immune organ already covers?`

It should also answer:

`Which constitutional need or truth function does it serve?`
