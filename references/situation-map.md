# Four-Need Situation Map

When explicitly invoked, structure the response like this:

- `自治判断`: what the agent must fully own and what truly remains human-only
- `全局最优判断`: how correctness, time, cost, interruption, reuse, and evolution are being balanced
- `能力复用判断`: what reusable skills, tools, scripts, APIs, or auth state already exist
- `验真判断`: what evidence must appear before claiming completion
- `进化判断`: what should be promoted after closure and what should be dropped
- `当前最大失真`: the most likely current distortion in the agent's model

This output should reveal internal judgment, not just external action sequencing.
